* Make sure all the code for simplepsychtoolbox is in your path and make sure the code in this file is in your path too, then run.

ttl_release_example();

* Once the above is run a disk will appear in the top right corner of your screen and remain there.  To advance to the next disk hit the '5' key on your keyboard.  

* The scanner sends a '5' for every TR (TTL pulse).  Every time you hit '5' in this experiment you are mimicking the scanner's signal.  

* There are 10 disks in total.
